# 开发指南

## 开发环境搭建
1. **安装 Node.js**
   ```bash
   # 使用 nvm 安装 Node.js
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
   nvm install 16
   nvm use 16
   ```

2. **安装项目依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

## 代码规范
1. **Vue 组件规范**
   - 使用 `<script setup>` 语法
   - 组件名使用 PascalCase
   - Props 定义使用 TypeScript 类型
   - 事件名使用 kebab-case

2. **样式规范**
   - 使用 SCSS 预处理器
   - 类名使用 BEM 命名规范
   - 避免使用 ID 选择器
   - 使用 CSS 变量定义主题色

3. **JavaScript 规范**
   - 使用 ES6+ 语法
   - 使用 const/let 替代 var
   - 使用箭头函数
   - 使用模板字符串
