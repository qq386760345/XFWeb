# Vue 3 项目

这是一个使用 Vue 3 + Vite 创建的项目。

## 项目设置

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run serve
```

## 技术栈

- Vue 3
- Vite
- Vue Router

## 项目结构

```
├── src/
│   ├── App.vue          # 根组件
│   └── main.js          # 应用入口
├── index.html           # HTML 模板
├── vite.config.js       # Vite 配置
└── package.json         # 项目配置
```
